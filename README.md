# ecdp-capability-workbench

This repository contains a GitHub Actions workflow for passing manual metadata inputs to the companion repository.

## Workflow inputs

When you run the workflow manually, you can provide:

- landing zone name (default: `bu01`)
- environment (default: `dev`)
- service domain (default: `sd001`)

## How to run it

1. Open the Actions tab in this repository.
2. Select the workflow named `Metadata Input Demo`.
3. Click `Run workflow`.
4. Use the defaults or change them as needed.
